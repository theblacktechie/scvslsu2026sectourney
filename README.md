# The Show vs The Standard
## South Carolina vs LSU · SEC Tournament Quarterfinals · Mar. 7, 2026

An interactive data visualization built for web and mobile covering the SEC Tournament Quarterfinal between South Carolina and LSU.

---

## What's Inside

**the-show-interactive.html**
A single self-contained HTML file. No dependencies, no build process, no external frameworks. Open it in any browser and it works.

---

## Sections

**Score**
Final score with quarter-by-quarter breakdown. South Carolina 83, LSU 77.

**Key Numbers**
Four stat callouts: 19-game win streak vs LSU, 7th straight SEC title game appearance, 5 of 6 stat categories won, 1 LSU category win (free throw percentage).

**Team Identity**
Season-defining stats for both programs with an OFF/DEF toggle. Covers points per game, field goal percentage, assists, points allowed, rebounds, and average margin of victory.

**Head to Head**
Animated bar chart comparison. Toggle between This Game stats and Season Averages. Winning side highlighted in team color, losing side muted.

**Key Performers**
Six players, three per team, with box score stats and shooting lines.

**The Verdict**
Closing summary in South Carolina garnet. Labeled The Show vs The Standard.
